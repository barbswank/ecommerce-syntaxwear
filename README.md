# SyntaxWear

O SyntaxWear é um e-commerce moderno e estiloso desenvolvido para a venda de tênis e calçados online. Este projeto foi desenvolvido como uma demonstração de habilidades em desenvolvimento front-end, com foco na criação de uma interface atraente, responsiva e intuitiva utilizando HTML e CSS.

## Funcionalidades

- **Design responsivo:** o layout se adapta a diferentes tamanhos de tela, proporcionando uma boa experiência em desktops, tablets e dispositivos móveis.
- **UI/UX moderna:** design limpo e moderno, com foco na usabilidade e na experiência do usuário.
- **Componentização com CSS:** o CSS é organizado em componentes, facilitando a manutenção e a reutilização dos estilos.
- **Exibição de produtos:** a página inicial conta com uma seção hero, categorias de produtos e uma grade para exibição dos produtos.
- **Inscrição na newsletter:** formulário de inscrição para receber novidades e atualizações.
- **Links para redes sociais:** links para perfis de redes sociais disponíveis no rodapé da página.

## Tecnologias utilizadas

- **HTML5:** utilizado para a estrutura e o conteúdo das páginas web.
- **CSS3:** utilizado para a estilização do projeto, com recursos modernos como Flexbox e CSS Grid.
- **Google Fonts:** utilizado para adicionar fontes personalizadas ao projeto.
- **Visual Studio Code:** utilizado como editor de código.
- **Git e GitHub:** utilizados para controle de versão e gerenciamento do repositório.

## Estrutura de pastas

O projeto está organizado da seguinte forma:

```text
ecommerce-syntaxwear/
│
├── css/
│   ├── components/
│   │   ├── footer.css
│   │   ├── header.css
│   │   ├── hero.css
│   │   ├── product-category.css
│   │   └── product-grid.css
│   │
│   ├── base.css
│   ├── reset.css
│   └── variables.css
│
├── images/
│   ├── banners/
│   ├── favicons/
│   ├── icons/
│   ├── logo/
│   └── products/
│
├── js/
│
├── index.html
└── README.md
```

## Descrição das pastas e arquivos

- **/css**: contém os arquivos globais de estilos, como reset, variáveis e base.
- **/css/components**: contém os estilos CSS específicos de cada componente do site, como header, footer, hero, categorias e grade de produtos.
- **/images**: contém todas as subpastas de imagens organizadas por categorias (banners, favicons, icons, logo e products).
- **/js**: contém os arquivos JavaScript do projeto. Atualmente, a pasta está vazia.
- **index.html**: arquivo HTML principal do projeto.
- **README.md**: documentação do projeto.

## Como executar o projeto

Para executar o projeto localmente, basta clonar o repositório e abrir o arquivo "index.html" no navegador.

1. Clone o repositório
```bash
git clone https://github.com/barbswank/ecommerce-syntaxwear.git
```

2. Acesse a pasta do projeto
```bash
cd ecommerce-syntaxwear
```

3. Execute o projeto
Abra o arquivo `index.html` diretamente no navegador ou utilize uma extensão como o *Live Server* no Visual Studio Code.
